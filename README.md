# Gym Pulse Exercise Database

## Overview
This repository contains a structured JSON database of **870+ gym exercises** with detailed attributes, including:

- **Exercise Name**
- **Force Type**
- **Difficulty Level**
- **Mechanics**
- **Equipment Used**
- **Primary and Secondary Muscles**
- **Instructions**
- **Exercise Category**
- **Associated Images**
